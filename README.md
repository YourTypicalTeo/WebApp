# Forum Web App

A lightweight forum-style web application where users can log in, create discussion topics, send messages, and manage their passwords. Built using Java Servlets and JSP.

- Java (Jakarta EE)
- JSP + Servlets
- MySQL (via XAMPP)
- HTML + CSS + JS (vanilla)
- Apache Tomcat (10+)
- Eclipse IDE
